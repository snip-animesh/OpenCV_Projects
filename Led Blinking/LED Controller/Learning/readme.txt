Project Name : Controlling Led with Hand Gesture.

It's my computer vision based project using python and openCV . 

Library Used : Mediapipe

Hardware used : Arduino Uno , Jumper wires , Led's , bread board.

Let's talk about this.

Five Fingers in our hand is dedicated to one led serially . 
According to which finger is open , that led will reamin on and the fingers which are closed that led's will remain off.
